# 추상화 끄적이기.

- 추상화가 필요한 이유?
- 문제를 단순화 시킴.
  - 이해가 쉬움.
- 추상화 방식?
- 인터페이스 사용
- 상속 과 구현 사용.

- 단일 책임 원칙 : 하나의 원칙만 가짐것
- 인터페이스 분리 원칙 : 여러기능을 다 인터페이스로 분리할것.

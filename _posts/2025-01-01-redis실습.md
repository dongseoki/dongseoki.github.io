## redis 가자.
```
docker pull redis:6.2
```
```
docker run --rm -it -d -p 6379:6379 redis:6.2
```

```
docker exec -it focused_kepler redis-cli
docker exec -it focused_kepler redis-cli monitor
```

```
docker exec -it focused_kepler redis-cli --stat
```
